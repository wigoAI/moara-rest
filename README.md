# moara-rest
- moara-core 를 활용한 기본적은 rest api 제공
- 내부망 서비스 입니다. 인증부분은 front에서 처리하고 이 프로젝트는 realtime in memory search analysis 를 제공합니다.
- spring boot 을 활용하여 엔진 기동시에 rest 서비스를 같이 기동할 수 있습니다.

# 개발환경
- open jdk 1.8
- spring boot 2.3.2
- moara-core last version
